TIP

  • 방 자동 새로고침(상단 노출) 기능은 자동으로 적용됩니다.

  • 무한디스 유발 추방: 인게임 내에 고의적으로 렉을 유발하는 유저가 존재하면 자동으로 추방됩니다.

  • 추방 투표: 과반수 이상이 찬성하면 해당 플레이어는 추방됩니다.
    [!투표 1~8] 해당 플레이어 추방 투표 시작 (1~8 = 플레이어 고유번호, 커스텀 뷰에서 확인)
    [!찬성] 해당 플레이어 추방 찬성 (전체채팅 필수)

  • 플레이어 색상 > 기본 색상 변경: 플레이어의 색상들이 P1~P8 기본 색상으로 변경됩니다. (적용 후 해제시에는 해당 게임이 끝나야 풀리게 됩니다.)

  • 소유자 뷰어: 플레이어의 유닛을 클릭하면 소유자의 닉네임을 표시합니다.

  • 체크된 옵션들만 적용됩니다.
